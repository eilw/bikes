# Bysykler

This project is bootstrapped with [Create Elm App](https://github.com/halfzebra/create-elm-app).

Krever node v16.13.0 eller lavere. Det feiler å kjøre i nyere versjoner av node.

For å kunne kjøre:

```shell
npm install create-elm-app -g

elm-app start
```
